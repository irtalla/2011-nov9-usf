Fizz Buzz!

A user will input a number from 1 - 100. Your code will print all of the numbers in order up to the input number with a few exceptions.

  • Any number divisible by 3 is replaced with FIZZ
  • Any number divisible by 5 is replaced with BUZZ
  • any number divisible by both is replaced with FIZZBUZZ
